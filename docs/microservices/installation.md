# Cài đặt môi trường

## Source code
- [https://gitlab.topworking.vn/hoangdo/kong-demo](https://gitlab.topworking.vn/hoangdo/kong-demo)

## Source code này có những gì?
- Kong Gateway.
- Keycloak đảm nhiệm vai trò AuthService.
- Sử dụng keycloak để Authentication cho Kong Gateway và các service phía sau.
- Một bản demo về workspaces service, chứng minh Kong gateway có thể authentication và service phía sau đã được secure, đồng thời có user data.
- Một bản demo dự án frontend (sử dụng react), implement quá trình đăng nhập sử dụng keycloak như SSO server, đồng thời request lấy data từ Kong gateway.
